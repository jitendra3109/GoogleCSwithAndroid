Welcome! This is an offline version of the Applied CS with Android course, in case of connectivity issues. If you have a reliable connection to the Internet, we strongly suggest using the online version to make sure you have access to all the links and the most up to date resources.

All the class resources have been made available offline in this version, as well as certain Google documentation. However, where we can't provide an offline copy, the external link remains.

You will also find the binaries for Android Studio in this package, as well as installation instructions to enable you to get started quickly.

Good luck!
