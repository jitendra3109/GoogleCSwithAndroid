Unit 2 Preparation (Self Study)

In this unit, we will explore in more details the structure of an Android app.

Start by doing the "build your first app" activity on the Android developer site (you can stop when you reach the "Build an Intent" section).

Then go back to the Anagram program from last unit's activity and take a look at its UI and the AnagramActivity class to understand how it is connected to the UI and how it is connected to the AnagramDictionary.

Implementing a UI

There will be no starter code in this unitâ€™s workshop so you will need to be able to create a simple UI for a blank activity. In the workshop, we will be implementing a turn-based dice game, so we will require a way to roll the die (and show the result), end the player turn and to reset the game.

Before you arrive at the workshop, create the UI shown in the image below using either the visual editor or the XML editor (or probably a combination of both).

The UI is composed of:

  • a TextView to display the score and status of the game
  • an ImageView to display the current die (default to the image of your choice from these dice faces)
  • three buttons to either roll the die, end your turn or start over

You will implement the logic behind the components in the workshop, but you may want to take some time now to consider the basic logic that may need to be implemented in a turn-based dice game (how might you roll? What will you need to do before and after every turn?) so you are ready in the workshop.
